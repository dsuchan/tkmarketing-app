## What is TKmarketing
TKmarketing is a simple SPA built with Vue 3.x, Vue Router 4.x, Vuex and Sass. It represents my girlfriends website to showcase her business.

## Used Technologies
- Vue ^3.2.13
- Vue Router ^4.0.3
- Vuex ^4.0.0
- Sass ^1.51.0
- Vue Toast Notification 3.0
- EmailJS ^3.6.2
